로그라이크 장르 기반 텍스트 게임

[필수 기능]
- 단순 행동 패턴 2가지 이상 구현 (공격, 도망 등)
- 플레이어 class에서 플레이어 스태이터스 관리 (공격력, 체력 등)
- 간단한 전투 로직 구현 (플레이어의 공격 -> 몬스터 피격 등)
- 스테이지 클리어 시 플레이어 체력 회복
- 스테이지 진행과 비례해서 몬스터의 체력과 공격력 증가

[필수 기능 가이드]
- 플레이어 행동 패턴 구현엔 switch 문을 활용해보자
- while문 사용 시 종료 조건이 꼭꼭 있어야함! (스테이지 클리어 조건, 전투 종료 조건, 게임 종료 조건 등)
- 확률 구현엔 Math.random() 메서드를 활용해보자
- 함수 호출 순서에 따라 변수와 객체들을 잘 관리해보자

[도전 기능]
- 확률 로직 적용
- 복잡한 행동 패턴 구현 (연속 공격, 방어하기 등)

- 도전 : 확률로직 구현 / 복잡한 행동패턴 구현 / 새로운기능 구현 / 전체적인 코드흐름
